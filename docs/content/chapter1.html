Estructuras iterativas (ciclos)#

Instrucción while#

  • Sintaxis:

    while < expresión de resultado booleano > :
  < bloque de una o más instrucciones con indentado >

  • Semántica:

    • Si la < expresión de resultado booleano > es verdadera, ejecutar el < bloque de una o más instrucciones con indentado >. Una vez terminada la ejecución del bloque de instrucciones, regresar el punto de ejecución del programa a la línea donde empezó la instrucción while

    • Cuando la < expresión de resultado booleano > sea falsa, continuar con la ejecución secuencial del programa inmediatamente después del < bloque de una o más instrucciones con indentado >.

Nota: Una forma alternativa de interpretar la instrucción while es: Mientras las condición sea verdadera, volver a ejecutar el bloque de instrucciones. A diferencia del if, la ejecución del bloque de instrucciones se sigue repitiendo mientras la condición sea cierta.

Ejemplo: Contador#

Escriba un programa que imprima los primeros \(n\) números naturales.

n = int( input( 'Ingrese el número n: ' ))

i = 1  # Variable que cumplirá el rol de contador

while i <= n:
  print(i)
  i = i+1  # Operación que incrementa el valor del contador en 1

print('Terminé!')

Ejercicios#

  1. Escriba un programa que imprima los primeros \(n\) pares (empezando desde 2).

  2. Escriba un programa que imprima los números: \(4, 9, 14, 19, 24, \dots , 49\)

  3. Escriba un programa que imprima \(n\) potencias de 2. Ejemplo: Para \(n=3\) se debe imprimir:
    \(2^0 = 1\)
    \(2^1 = 2\)
    \(2^2 = 4\)

Ejemplo: Acumulador#

Escriba un programa que calcule la suma de los primeros \(n\) números naturales.

n = int( input( 'Ingrese el número n: ' ))

suma = 0  # acumulador
i = 1  # contador

while i <= n:
  suma = suma + i  # acumular en suma los valores de i
  i = i+1  # incrementar el contador para producir el siguiente número natural

print('La suma es: ', suma)

Ejemplo: Validación de insumo#

Explique cómo se comporta el programa dependiendo del valor que se ingrese.

horas = int( input('Ingrese hora [1...12]: ') )

while horas < 1 or horas > 12:
  horas = int( input('Vuelva a ingresar hora [1...12]: ') )

Asignaciones con operador#

En ejemplos anteriores, para incrementar el valor de los contadores, se usó instrucciones del tipo:
i = i+1

Se puede lograr el mismo efecto usando una asignación con operador aritmético:
i += 1

En general lo operadores aritméticos: + - * / // % ** se pueden combinar con asignaciones. Con lo que una instrucción como:
i = i*2
Se puede escribir también como:
i *= 2

  1. Diga cuantas veces se ejecutará el while del siguiente código:

i = 0

  print(i)

  1. Usando while encuentre el resultado de: \(3+6+9+12+\dots+60 = \sum_{i=1}^{20}(3i)\)

  2. Usando while encuentre el resultado de: \(1\cdot 2\cdot 3\cdot 4 \dots 15 = \prod_{i=1}^{15}(i)\)
